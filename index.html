WhatUser

In the game "WhatUser," the objective is to input a random username into the provided text box and receive points based on the system's detection of the user's status. The game operates as follows: Input a Random Username: To begin, the player must enter a completely random username into the designated text box. This username can consist of any combination of letters, numbers, or symbols, but it must be entirely arbitrary to maintain the randomness of the game. User Detection: The system will then perform a real-time check to determine the status of the entered username. There are two possible outcomes: a. User Exists and is Active: If the system detects that the username exists and is not deleted or banned, the player will earn one point. c. User Doesn't Exist: If the system cannot find any record of the entered username, the player will lose one point. Scoring: The player's score will be calculated based on the outcomes mentioned above. The game can be played for a set number of rounds or until a specified time limit is reached. The goal of "WhatUser" is to accumulate as many points as possible by accurately guessing the status of randomly generated usernames. It requires a combination of luck and deduction, making it an engaging and challenging game for users.
